Reveal-Based Model

Define all content upfront, choreograph when parts appear

Data Structure

type Slide = {
  // All content for this slide — defined upfront
  content: (step: number) => React.ReactNode;

  // Graphic states per step within this slide
  steps: {
    graphic?: Partial<GraphicState>;
    transition?: TransitionConfig;
  }[];
};

The <Reveal> Component

// Shown when currentStep >= step
function Reveal({ step, children, ...motionProps }) {
  const currentStep = useSlideStep();
  if (currentStep < step) return null;
  return (
    <motion.div
      initial={{ opacity: 0, y: 8 }}
      animate={{ opacity: 1, y: 0 }}
      {...motionProps} // override defaults
    >
      {children}
    </motion.div>
  );
}

Example Slide

{
  content: (step) => (
    <>
      <h2>The Rounded Corner</h2>

      <Reveal step={0}>
        <p>CSS border-radius draws a simple circular arc.</p>
      </Reveal>

      <Reveal step={1}>
        <p>Notice the abrupt transition point where
        the curve meets the straight edge.</p>
      </Reveal>

      <Reveal step={2} initial={{ scale: 0.9 }} animate={{ scale: 1 }}>
        <div className="callout">
          Arcs are segments of perfect circles — they can't
          taper smoothly into a straight line.
        </div>
      </Reveal>
    </>
  ),

  // 3 steps → 3 graphic states
  steps: [
    { graphic: { showRounded: true } },
    { graphic: { showMeasurement: true } },
    { /* no graphic change */ },
  ],
}

Navigation

Next (within slide)

Increments step counter. New <Reveal> elements appear. Graphic animates to step's state.

Next (last step)

Advances to next slide. Content panel transitions out/in via AnimatePresence.

Back

Decrements step (or goes to prev slide's last step). Graphic state replays to match.

Graphic State Across Slides

Graphic state carries forward between slides (as you requested). Each step's graphic merges into the running state. Going backward replays all steps from 0 to recompute the cumulative state.
