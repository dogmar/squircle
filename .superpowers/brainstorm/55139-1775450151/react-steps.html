Steps with React Content

Each step defines arbitrary React nodes with animation control

Data Structure

type Step = {
  // The text panel content — full React nodes
  content: React.ReactNode;

  // Graphic state to merge (same as before)
  graphic?: Partial<GraphicState>;

  // How the graphic animates to this state
  transition?: TransitionConfig;
};

Example Step Definitions

const steps: Step[] = [
  {
    content: (
      <div>
        <h2>The Rounded Corner</h2>
        <motion.p
          initial={{ opacity: 0, y: 10 }}
          animate={{ opacity: 1, y: 0 }}
        >
          CSS border-radius draws a circular arc...
        </motion.p>
      </div>
    ),
    graphic: {
      showRounded: true,
      showSuperellipse: false,
      showCorrected: false,
      amount: 1.5,
    },
  },
  {
    content: (
      <motion.div
        initial={{ opacity: 0 }}
        animate={{ opacity: 1 }}
      >
        <p>Now overlay the superellipse...</p>
        <p className="text-sm text-gray-400">
          Notice the gap between the curves
        </p>
      </motion.div>
    ),
    graphic: { showSuperellipse: true },
    transition: { duration: 0.8 },
  },
];

How It Works

Content Panel

Each step's content fully replaces the text panel. The content itself manages its own enter animations via Framer Motion components.

Graphic State

Same partial-merge model. graphic merges into running state. Framer Motion animates the SVG elements to match.

AnimatePresence

Content wrapped in <AnimatePresence mode="wait"> so outgoing content can animate out before incoming content animates in.

Full Freedom

Steps can include images, math formulas, styled components, interactive mini-demos — anything React can render.
