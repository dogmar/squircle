Refined: Slides with Sub-Steps

Each slide has ordered steps — clicking "Next" reveals the next step or advances to the next slide

Data Structure

const slides: Slide[] = [
  {
    title: "The Rounded Corner",
    steps: [
      // Step 1: Show the rounded corner
      {
        text: "CSS border-radius draws a circular arc.",
        graphic: { showRounded: true },
      },
      // Step 2: Reveal more info (same slide)
      {
        text: "Notice the abrupt transition where
              the curve meets the straight edge.",
        graphic: { showMeasurement: true },
      },
    ],
  },
  {
    title: "Enter the Superellipse",
    steps: [
      {
        text: "The superellipse is a smoother curve...",
        graphic: { showSuperellipse: true },
      },
      {
        text: "But look — the perceived radius is smaller!",
        graphic: { measureArc: "superellipse" },
      },
      // Step 3: Animate K value sweep
      {
        text: "Watch what happens as we increase K...",
        graphic: { amount: 3 },
        transition: { duration: 1.5 },
      },
    ],
  },
];

How It Works

Click "Next"

If more steps remain in the current slide → reveal next step's text, animate graphic to new state. Previous steps' text stays visible.

Last Step → Next Slide

When all steps exhausted, "Next" transitions to the next slide entirely — title changes, text resets, graphic animates to new base state.

Graphic State Merges

Each step's graphic is a partial — it merges into the running state. Step 2 only needs to specify what changes.

Transition Control

Optional transition per step controls animation duration/easing. Default is a smooth 0.5s ease. K value sweeps get longer durations.

State Accumulation Model

Slide 1, Step 1: { showRounded: true }
Slide 1, Step 2: { showRounded: true, showMeasurement: true } ← merged
Slide 2, Step 1: { showRounded: true, showSuperellipse: true } ← fresh base from slide
Slide 2, Step 2: { ...prev, measureArc: "superellipse" } ← merged
Slide 2, Step 3: { ...prev, amount: 3 } ← animated over 1.5s
