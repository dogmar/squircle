Flat Step Model

A single ordered list of steps. Each step declares what changes.

Data Structure

type Step = {
  title?: string; // New title (clears text panel)
  text?: string; // Text to append to the panel
  graphic?: Partial<GraphicState>; // Merges into running state
  transition?: {
    duration?: number; // seconds (default 0.5)
    ease?: string; // easing function
  };
};

type GraphicState = {
  showRounded: boolean;
  showSuperellipse: boolean;
  showCorrected: boolean;
  amount: number;
  showMeasurement: boolean;
  measureArc?: "rounded" | "superellipse" | "corrected";
};

Example Sequence

0
title: "The Rounded Corner"   text: "CSS border-radius..."   graphic: showRounded: true
1
text: "Notice the abrupt join..."   graphic: showMeasurement: true
2
title: "The Superellipse"   text: "A smoother curve..."   graphic: showSuperellipse: true
3
text: "But the perceived radius shrinks!"   graphic: measureArc: "superellipse"
4
text: "Watch as we increase K..."   graphic: amount: 3   transition: 1.5s
5
title: "The Correction"   text: "We can compensate..."   graphic: showCorrected: true, amount: 1.5   transition: 1s

Rules

title present?

Clear the text panel, set new title. This is a "chapter break."

text present?

Append to the text panel (fade in). Accumulates within a chapter.

graphic present?

Merge into running graphic state. Animate to new values using Framer Motion.

Going backward?

Recompute state by replaying steps 0..N from scratch. Animate to the result.
