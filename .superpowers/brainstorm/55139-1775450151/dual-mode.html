Dual-Mode Content: Append vs Replace

Each step chooses whether to add to the panel or replace it

Updated Step Type

type Step = {
  content: React.ReactNode;
  mode: "append" | "replace";
  graphic?: Partial<GraphicState>;
  transition?: TransitionConfig;
};

Example Flow

0 replace 🔄 <h2>The Rounded Corner</h2><p>CSS border-radius...</p> 1 append ➕ <p>Notice the abrupt join...</p> — added below step 0's content 2 append ➕ <p>This is by design — arcs are circles...</p> — added below step 1 3 replace 🔄 <h2>The Superellipse</h2><p>A smoother curve...</p> — clears panel 4 append ➕ <p>But the perceived radius shrinks!</p>

Panel Behavior

mode: "replace"

Previous content animates out via AnimatePresence, then new content animates in. Clean slate.

mode: "append"

New content animates in below existing content. Previous content stays in the DOM. Panel scrolls if needed.

Going Backward

Replay steps 0..N to reconstruct which content nodes are visible. All "replace" steps clear earlier nodes. All "append" steps add to the list. The graphic state is also replayed from scratch.
